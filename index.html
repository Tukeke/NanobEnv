CoPIM by XayOn

Console PIM. Copim is an advanced console personal information manager, with almost all the functionalities of the big GUI-driven PIM' software out there

CoPIM uses all the power of mutt, irssi, wyrd (now with remind goodies!), abook, and SIP software (as skype) bash and screen to make a great personal information manager with voip support available on non-gui environments.

Dependencies

Skype (Optional), mutt, irrsi, wyrd, abook, bash, screen, python

Install

Standard makefile way =)
